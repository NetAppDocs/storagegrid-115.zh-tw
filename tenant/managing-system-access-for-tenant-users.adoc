= 管理租戶使用者的系統存取
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從聯盟身分識別來源匯入群組並指派管理權限、藉此授予使用者存取租戶帳戶的權限。您也可以建立本機租戶群組和使用者、除非整個StorageGRID 整個版的整套系統都有單一登入（SSO）功能。
